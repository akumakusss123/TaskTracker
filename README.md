## Описание проекта
TaskTracker — консольный менеджер задач.  
Позволяет:  
- Добавлять, редактировать и удалять задачи  
- Просматривать список задач  
- Изменять статус задачи  
- Хранение данных в JSON или в памяти  
- Логирование действий  

## Как запустить
1. Открыть решение TaskTracker.sln  
2. Выбрать **TaskTracker.App** как стартовый проект  
3. Запустить (F5)  
4. При старте выбрать режим хранения: JSON или Memory
